## Symbols I'm always looking for

| Symbol | Keys | Description |
|--|--|
| É | <kbd>Alt Gr</kbd> + <kbd>?</kbd> release, then <kbd>E</kbd> | Uppercase E with acute accent |
| È | <kbd>Shift</kbd> + <kbd>^</kbd> release, then <kbd>E</kbd> | Uppercase E with grave accent |
| · | <kbd>Alt Gr</kbd> + <kbd>.</kbd> | Middle dot (*point médian*) |
| ¡ | <kbd>Alt Gr</kbd> + <kbd>Shift</kbd> + <kbd>1</kbd> | Inverted exclamation mark |
| ¿ | <kbd>Alt Gr</kbd> + <kbd>Shift</kbd> + <kbd>?</kbd> | Inverted question mark |

## Displaying the keyboard map
```bash
gkbd-keyboard-display -l "ch(fr)"
```
